### Spooner Mode
Spooner mode lets you pick up, rotate, and place objects manually. Enable this mode to adjust the position of items you have spawned. This is useful for creating scenes or building custom structures in the game.

### World Editing
Modify the environment settings. Change the time of day, current weather, or gravity levels. These settings update in real-time as you move through the world.

### Entity Management
Track NPCs and vehicles in your immediate area. You can delete items, heal characters, or repair vehicles that you have interacted with during gameplay.

## 🛡️ Troubleshooting

If the menu does not appear when you press F8, verify these points:

*   Confirm that the files reside in the same folder as `GTA5.exe`.
*   Ensure that you have installed the core library known as Script Hook V.
*   Check that your antivirus software did not block the file. Some security programs flag new tools as suspicious. Create an exclusion for the game folder if the tool fails to load.
